# 🛡️ ScamShield
### AI-Assisted Cyber Safety & Scam Awareness Platform

> Protect Yourself From Online Scams.

SafeNet AI is a futuristic cybersecurity awareness platform designed to educate users, analyze suspicious scams, and improve digital safety through AI-assisted explanations, multilingual accessibility, and interactive learning experiences.

Built during **HACKHIVE-2k26** for the **Cyber Safety & Awareness Hub** challenge.

---

# 🚀 Overview

With the rapid rise of digital fraud, phishing attacks, UPI scams, fake job offers, and OTP theft, millions of users lack accessible cybersecurity education.

SafeNet AI focuses on:
- scam awareness
- prevention
- education
- community-driven reporting

instead of only automated detection systems.

The platform combines:
- scam analysis
- cybersecurity education
- multilingual accessibility
- fraud reporting
- interactive scam awareness experiences

into one modern cyber defense platform.

# 🧠 Threat Analysis System

SafeNet AI uses a hybrid scam analysis approach combining:
- rule-based threat detection
- suspicious keyword analysis
- URL safety checks
- AI-generated educational explanations

The platform focuses primarily on:
- cyber awareness
- prevention
- scam education
- user-friendly explanations

rather than building a proprietary cybersecurity AI model.

```txt

   ↓
```

# ⚡ Installation

## Clone Repository

```bash

```

# 🔧 Setup

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Application runs on:

```txt
http://localhost:5173
```
